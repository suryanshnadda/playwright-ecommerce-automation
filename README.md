# Playwright ecommerce automation
This repository contains an end-to-end automation test suite for an e-commerce application built using Playwright. It validates core customer journeys such as product search, cart operations, checkout flow, order confirmation, and UI behavior.

The framework is designed to be clean, modular, and scalable, following recommended Playwright patterns and page-object architecture. It is suitable for learning automation, demonstrating best practices, or serving as a foundation for a real-world QA workflow.

Key Features

Login Automation
Validates secure user authentication and ensures the login flow functions reliably.

End-to-End Checkout Verification
Covers the complete purchase journey, including product selection, cart operations, and order confirmation.

Automated Screenshot Capture
Generates screenshots after each test case to support debugging, traceability, and reporting.

Optimized and Maintainable Codebase
Structured using best practices to ensure readability, scalability, and efficient test execution.
